# 🍷 AI Wine Predictor: The Lab
**A Progressive Web App (PWA) for Personalized Wine Curation**

[![PWA Shield](https://img.shields.io/badge/PWA-Ready-green)](https://web.dev/progressive-web-apps/)
[![Firebase Shield](https://img.shields.io/badge/Database-Firestore-orange)](https://firebase.google.com/)

## 📖 Overview
The **Wine Lab** is a mobile-first application designed to solve the "I forgot if I liked this" problem. By combining a personal relational database with AI-driven sentiment analysis, the app helps users track, rate, and predict their enjoyment of wines from around the world.

This project was developed as part of my **Associate of Science in AI and Machine Learning** at Palm Beach State College.

## ✨ Key Features
* **PWA Integration:** Fully installable on Android and iOS with offline support.
* **Real-Time Database:** Powered by **Firebase Firestore** for instant syncing across devices.
* **AI Insights:** Uses sentiment analysis and behavioral data to predict if you will enjoy a bottle before you buy it.
* **Global Tracking:** Record memories from local spots in Florida to hidden gems in Budapest and beyond.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Mobile-First Design), JavaScript (ES6+).
* **Backend/Database:** Firebase Firestore.
* **Deployment:** GitHub Pages.
* **Tooling:** PWABuilder (for Android App Bundle generation).
