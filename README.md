# 🍷 Wine Lab Predictor
A mobile-first, data-driven application to predict wine preferences for Colleen and her boyfriend.

## 🛠 Project Standards
- **Mobile-First:** Responsive design for use at Trader Joe's/Doris.
- **Accessibility:** WCAG 2.1 AA Compliant (High contrast, semantic HTML).
- **Data-Driven:** Uses Schema.org `Product` extensions.
- **i18n:** Built to support English and Italian.

## 📊 Data Schema
We categorize wine based on four "Predictor Variables":
1. **Grape Type:** Primary flavor profile (e.g., Sangiovese).
2. **Location:** Terroir indicators (e.g., Tuscany, Italy).
3. **Vintage:** Impact of weather on acidity/tannins.
4. **Process:** Production methods (e.g., Stainless Steel = Clean; Oak = Smoky).
